Welcome to C#

Introduction

C# pronounced as 'C-sharp', is an elegant object-oriented language that enables developer to build a variety of secure and robust applications that runs in the .Net Framework.
By the help of C#, we can developed different applications like:

  • Windows applications
  • Web services
  • Mobile applications
  • Client-server applications
  • Distributed applications
  • Database applications and much more.

The .Net Framework

The .Net framework is consist of the Common Language Runtime (CLR) and the .Net Framework library.
The CLR is the foundation of the .Net framework. It manages code at execution time, providing core services such as memory management, code accuracy, and many other aspect of your code.
The class library is a collection of classes, interfaces, and value types that enable you to accomplish a range of common programming tasks, such as data collection, file access, and working with text.
C# programs use the .Net Framework class library extensively to do common tasks and provide various functionalities.

History

Anders Hejlsberg is known as the founder of C# language, and its current version is 6.0 and is provided and maintained by Microsoft.
C# is based on C++ and Java, but it has many additional extensions used to perform component oriented approach.
C# has evolved much since its first release in the year 2002. It was introduced with .Net Framework 1.0.

Features

  1. Simple
    2. It provides structured approach (to break the problem into parts), rich set of library functions, data types, etc.
  2. Modern programming language
    3. It is based on the current trend and it is very powerful and simple building scalable, interoperable and robust applications.
  3. Object-oriented
    4. OOPs makes development and maintenance easier where as in Procedural-oriented language which is not easy to manage when the application grows.
  4. Type safe
    5. C# type safe code can only access the memory location that it has permission to execute. Therefore, it improves a security of the program.
  5. Interoperability
    6. It enables programs to do almost anything that a native C++ application do.
  6. Scalable and Updateable
    7. It's automatic scalable and updatable language. For updating our application we delete the old files and update them with new ones.
  7. Component oriented
    8. It's predominant software development methodology used to develop more robust and highly sclable applications.
  8. Structured programming language
    9. C# is a structured language in the sense that we can break the program into parts using functions. So, it is easy to understand and modify.
  9. Rich Library
    10. It provides a lot of in-built functions that makes the development fast.
  10. Fast speed
    11. The compilation and execution time is fast.
