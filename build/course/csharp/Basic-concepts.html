Chapter 1 : Basic Concepts

1.1 Variable

Programs typically use data to perform tasks. Creating a variable reserves a memory location, or a space in memory, for storing variables. It is called variable because the information stored in that location can be changed when the program is running. To use a variable, it must be declared by specifying the name and data type. A variable name, also called an identifier, can contain letters, numbers, and the underscore character '_', and must start with a letter or underscore.
Although the name of a variable can be any set of letters and numbers, the best identifier is descriptive of the data it will contain. This is very important in order to create, understandable and readable.

firstName and lastName are good descriptive variable names. While, abc and xyz are not.

1.2 Variable Types

A data type defines the information that can be stored in a variable, the size of needed memory and the operations that can be performed with the variable.

Variable Type Example
Decimal types decimal
Boolean types true or false value, as assigned
Integral types int, char, byte, short, long
Floating types float and double
Nullable types Nullable data types

For example, to store an integer value (a whole number) in a variable, use the int keyword:

int myAge;

The code above declares a variable myAge of type integer.

A line of code that completes an action is called a statement. Each statement in C# must end with a semicolon.

You can assign the value of a variable when you declare it:

int myAge = 18;

or later in your code:

int myAge;
myAge = 18;

Remember that you need to declare the variable before using it.

1.3 Data Types

There are 3 types of data types in C# language.

Type Data Type
Value Data Type int, float, double, char, bool, string, etc.
Reference Data Type String, class, object, and interfaces
Pointer Data Type Pointers

1.3.1 Value Data Type

The value data type are integer-based and floating-point based type that supports both signed and unsigned literals.

There are 2 types of value data type.

  • Predefined Data Types - such as integer, boolean, float, etc.
  • User defined Data Types - such as structure, enumerations, etc.

There are a number of build-in data types in C#. The most common are:

  • int - used to declare an integer value.
  • float - floating point number.
  • double - double-precision version of float.
  • char - a single character.
  • bool - Boolean that can have only one of two values; true or false.
  • string - A sequence of characters.

The statements below use C# data types:

int num = 14;
double pi = 3.1415;
char letter = 'R';
bool isValid = true;
string fullName = "Susey Santos";

Note that char values are assigned using single quotes and string values require double quotes.

1.3.2 Reference Data Type

The reference data types do not contain the actual data stored in a variable, but they contain a reference to the variables. If the data is changed by one of the variables, the other variable automatically reflects this change in value.

There are 2 types of reference data type in C# language.

  • Predefined Types - such as objects, String.
  • User Defined Types - such as classes, interface.

1.3.3 Pointer Data Type

The pointer in C# language is variable, It is also known as locator or indicator that points to an address of a value.

Symbol used in pointer
Symbol Name
& (ampersand sign) Address operator
* (asterisk sign) Indirection operator

Declaring a pointer

The pointer in C# language can be declared using * (asterisk symbol).

int * num; // pointer to int
char * letter; // pointer to char
